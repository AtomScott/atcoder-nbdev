## A - Century
---

配点 : $100$ 点

### 問題文

西暦 $N$ 年は何世紀ですか？

### 制約

- $1 \le N \le 3000$

---


### 入力

入力は以下の形式で標準入力から与えられる。

```
$N$

```

### 出力

答えを整数として出力せよ。

---


### 入力例 1

```
2021

```

### 出力例 1

```
21

```

今年、西暦 $2021$ 年は $21$ 世紀です。

---


### 入力例 2

```
200

```

### 出力例 2

```
2

```

西暦 $200$ 年は $2$ 世紀です。
